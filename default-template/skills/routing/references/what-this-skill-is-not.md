## What this skill is NOT

- Not a permission system. The user can do any task in any profile.
- Not a hard routing rule. Default can still do website work if the user asks.
- Not a substitute for the user knowing their own profile structure. They set the profiles; they decide when to switch.

It's just a gentle reminder, surfaced early in the conversation, so the user can route work to the profile that has the right context and skills.
